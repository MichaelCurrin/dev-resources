### Installation

```sh
$ npm i -D jsdoc
```


## Usage

If you have it installed in your project or globally, or even if not installed at all.

```sh
$ npx jsdoc foo.js
```

By default, output goes to `out`.

Override with `-d OUT_DIR`.
