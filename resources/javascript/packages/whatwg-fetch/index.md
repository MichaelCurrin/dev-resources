title: whatwg-fetch
description: A browser polyfill for `fetch`

key_links:
  pkg_url: https://www.npmjs.com/package/whatwg-fetch
  
---

Note the docs say this only works in the browser and not in Node (server-side).

## Installation

```sh
$ npm install whatwg-fetch
```

## Usage
```javascript
import 'whatwg-fetch'

window.fetch(url)
```
